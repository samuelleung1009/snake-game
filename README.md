# 🎮 贪食蛇游戏合集 (Snake Game Collection)

这是一个包含两个版本的贪食蛇游戏仓库：
- **电脑版** (`snake_game.html`): 使用键盘方向键控制
- **手机版** (`snake_game_mobile.html`): 使用触摸手势控制

## 🚀 快速开始

### 方式一：直接访问
访问 [https://samuelleung1009.github.io/snake-game/](https://samuelleung1009.github.io/snake-game/)

### 方式二：下载到本地
```bash
git clone https://github.com/samuelleung1009/snake-game.git
cd snake-game
# 用浏览器打开任意HTML文件
```

## 🎮 游戏版本介绍

### 电脑版贪食蛇 (`snake_game.html`)
- **操作方式**: 键盘方向键 (↑ ↓ ← →)
- **特色功能**:
  - 现代化渐变界面设计
  - 最高分记录和保存
  - 暂停/继续功能
  - 自动保存最高分到浏览器

### 手机版贪食蛇 (`snake_game_mobile.html`)
- **操作方式**: 触摸手势（上下左右滑动）
- **特色功能**:
  - 专为竖屏优化设计
  - 触摸手势控制（主控制方式）
  - 备用方向按钮（如果手势不好用）
  - 震动反馈（设备支持时）
  - 初次使用手势引导

## 📱 使用说明

### 在电脑上
1. 下载HTML文件
2. 用浏览器打开
3. 使用方向键控制蛇的移动
4. 空格键暂停/继续

### 在手机上
1. 下载HTML文件到手机
2. 用浏览器打开
3. **推荐横屏游玩**
4. 在游戏区域滑动控制方向

### 直接访问（推荐）
直接访问GitHub Pages链接，无需下载任何文件。

## 🔧 技术特点

- 纯HTML5 + CSS3 + JavaScript
- 无外部依赖，完全离线运行
- 响应式设计，适应各种屏幕尺寸
- 使用localStorage保存最高分
- 性能优化，流畅的游戏体验
